### KBC Quiz Game (Kaun Banega Crorepati – CLI Version)
A command-line quiz game inspired by the popular television format.

**Features:**
- 15 structured multiple-choice questions  
- Progressive prize ladder system  
- Safe milestone levels (₹10,000 and ₹3,20,000)  
- Answer locking confirmation system  
- Quit option before answering  
- Input validation using loops  
- Clean modular workflow (separate functions for instructions, question display, answer handling, and prize logic)  
- Refactored control flow (replaced nested recursion with structured loops for better readability and stability)

This project focuses on structured program design, logical branching, loop control, and user interaction handling.

### Number Guessing Game (CLI Version)

An interactive command-line game where the player attempts to guess a randomly generated number within a limited number of attempts.

**Features:**
- Difficulty selection (Easy, Medium, Hard)
- Adjustable number range based on difficulty
- Fixed attempt limit with clear attempt tracking
- Smart hint system (Very Close / Close / Too High / Too Low)
- Input validation and error handling
- Replay option after game completion
- Structured modular design using functions

This project focuses on loop control, conditional logic, user input validation, and structured program flow while maintaining beginner-friendly implementation.
